Lab
Research Lab: USC Intelligence and Knowledge Discovery Lab
Year: 2020
Role: AI Research Assistent
This summer, I worked with INK Lab to begin explaining prior research done at the lab which looked at using neural networks as an NLP application. Neural network models are commonly adopted for sentiment classification, but in doing so, the “rationale” behind their predictions and classifications are often not well understood.
PROJECT GOAL

My goal this summer was to create a web-based application to visualize and demonstrate work that has done to “explain” the algorithms behind the proposed neural network model. By doing so, other researchers and users of this model can see the visual breakdown of sentence classification, using a hierarchical type of explanation.

ABOUT PROJECT

I built the front-end application of the web demo using Flask. We ran into some issues with the Flask framework and implementing the models’ arguments, but we’ve been able to provide word-level explanations back to the client so far.

At the lab, we trained and ran experiements on 2 common language models, LSTM and BERT, for sentiment analysis and relation extraction. Using the models, we then used the Sampling and Occlusion algorithm to explain and visualize the trained results. In our demo, we chose to use LSTM and BERT models because from previous research, we found that LSTM and BERT models show significantly higher human evaluations scores than other models.

Next Step

Our next step is to provide a web-based visualization of hierarchical explanations.Ultimately, we want to show how sentiment prediction builds up from atomic words to the full sentence, rather than looking at each word independently. By doing so, we can begin to improve human trust with these neural network models.
